Welcome to Browser Go!

If this is your first time playing Go, please see this great tutorial video.

To begin a game enter player names and ranks above, then click "Suggest Komi" Browser Go will calculate the appropriate komi based on AGA guidelines. To override Browser Go's suggestion, use the sliders above. Be sure to check the 'rank certainty' box if you're club-rated.

For now, Browser Go only supports games on a small board. I've got great things planned for the future, though! Lookout for new releases on the GitHub page!
